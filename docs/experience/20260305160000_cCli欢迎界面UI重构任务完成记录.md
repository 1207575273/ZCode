## UI 行为说明

| 时机 | 显示内容 |
|------|---------|
| 启动时 | 双栏欢迎界面（WelcomeScreen）|
| 发送第一条消息后 | 切换为聊天视图（ChatView）|
| 聊天中 | 消息历史向上堆叠，`> 你` 绿色，`◆ cCli` 青色 |
| 任何时候 | 底部 `❯` 输入框始终可见 |
| 输入 `/exit` 或 `/quit` | 优雅退出 |
