# ZCli 欢迎界面 UI 重设计

> 日期: 2026-03-05 | 参考原型: `docs/res/01_claude_原型.png`

---

## 目标

将现有 CLI 启动界面重构为与 Claude Code 风格一致的欢迎屏，首次输入后切换为聊天视图。

---

## 组件结构

```
App
├── WelcomeScreen    ← started=false 时显示
├── ChatView         ← started=true 时显示
└── InputBar         ← 始终在底部，两个 screen 共用
```

- `App` 持有 `started: boolean` 和 `messages: ChatMessage[]`
- 用户提交第一条消息 → `setStarted(true)` + 追加到 messages
- `StatusBar` 移除（原型中无此元素）

---

## 视觉布局

### WelcomeScreen

```
┌─ ZCli v0.1.0 ──────────────────────────────────────────┐
│                        │                                │
│  Welcome back!         │  Tips for getting started      │
│                        │  输入 /help 查看命令列表       │
│   ▄██████▄             │                                │
│   █ ■  ■ █             │  Recent sessions               │
│   █  ──  █             │  No recent activity            │
│   ▀██████▀             │                                │
│                        │                                │
│  claude-opus-4-6       │                                │
│  anthropic             │                                │
│  D:\path\to\cwd        │                                │
└────────────────────────┴────────────────────────────────┘
```

- 外边框使用 `borderStyle="round"`，标题 `ZCli v0.1.0` 嵌入顶部边框
- 左右两栏通过 `flexDirection="row"` + 中间竖线分隔
- 左栏宽度固定，右栏 `flexGrow={1}`

### InputBar

```
 ────────────────────────────────────────────────────────
 ❯ Try "how does <filepath> work?"
```

- 无 border box，仅一行 `─` 分隔线 + `❯` 提示符
- 使用 `ink-text-input` 处理输入

## 配色方案

| 元素 | Ink 属性 |
|------|---------|
| 外边框 + 版本标题 | `color="red"` |
| 右栏小节标题 | `color="yellow"` bold |
| "Welcome back!" | `color="white"` bold |
| ASCII 机器人 | `color="red"` |
| 模型名 / Provider | `color="white"` |
| CWD 路径 | `dimColor` |
| 输入提示符 `❯` | `color="green"` |
| 占位文字 | `dimColor` |
| 用户消息标签 `> 你` | `color="green"` bold |
| AI 回复标签 `◆ ZCli` | `color="cyan"` bold |

## 状态流

```
启动
  └─ App: started=false
       └─ 渲染 WelcomeScreen + InputBar

用户提交第一条消息
  └─ App: started=true, messages=[{role:'user', content}]
       └─ 渲染 ChatView + InputBar

后续消息
  └─ messages 追加，ChatView 更新
```
