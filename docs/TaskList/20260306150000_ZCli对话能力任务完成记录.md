### T8: useChat Hook
**Commit:** `f53e2c2`

**新增 `src/ui/useChat.ts`**

流式状态：
```typescript
messages: ChatMessage[]          // 历史（已完成）
streamingMessage: string | null  // 流入中的文字，null = 空闲
isStreaming: boolean             // 输入框禁用标志
error: string | null             // 错误信息
```

流式生命周期：
```
submit(text)
  → isStreaming=true, streamingMessage=''（空气泡）
  → 每个 text chunk: streamingMessage += chunk（打字机）
  → done: messages.push(assistantMsg), streamingMessage=null
  → 出错: error=msg, streamingMessage=null
```

中断支持：`abort()` 调用 `AbortController.abort()`

## 架构决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| LLM SDK | LangChain (`@langchain/anthropic` + `@langchain/openai`) | 项目核心理念，统一抽象 |
| GLM 接入方式 | `OpenAICompatProvider` + `baseURL` | GLM 兼容 OpenAI 协议，零额外代码 |
| baseURL | `open.bigmodel.cn`（非 `.ai`）| 以实际 curl 示例为准 |
| 工具调用 | 本期不做 | YAGNI，先跑通对话主链路 |
| Agent Loop | `useChat` hook 内联 | 避免过度设计，下期加工具时再拆 |
